# W4M ISA extractor
Version: 1.1.3

## Short description

<!-- 
This should only be 20 to 40 words, hopefully a single sentence.
-->

ISA data files extractor.

## Description

This tools is used to extract collections of data files (mzML, mzXML, nmrML, mzData or netCDF) from an ISA archive.

- ISA conversion.
